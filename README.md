# Harmonia Music - Clean Deployable Project

This package is a cleaned, Vercel-ready version of your app with:
- React frontend (Create React App structure)
- Vercel serverless API routes under /api
- Supabase-ready backend functions (store SUPABASE_URL and SUPABASE_SERVICE_KEY in Vercel env)

HOW TO DEPLOY:
1. Push this repository to GitHub.
2. On Vercel, import the repo.
3. Set environment variables in Vercel:
   - REACT_APP_ACCESS_TOKEN
   - REACT_APP_SHARED_LINK
   - SUPABASE_URL
   - SUPABASE_SERVICE_KEY
4. Deploy.

Note: run `npm install` locally before `npm start` or `npm run build`.
